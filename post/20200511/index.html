NOVOPlasty 组装线粒体全序列

2020-05-11 STUDY

😎学习笔记 NOVOPlasty
👍使用 NOVOPlasty 脚本组装线粒体全序列

NOVOPlasty 是一个perl脚本，在windows系统上运行之前要安装Perl编译器。

下载连接（Perl）

https://www.activestate.com/products/perl/downloads/

下载连接（脚本）

https://github.com/ndierckx/NOVOPlasty

使用方法：
1.perl的安装

直接双击运行下载的文件，按步骤安装。

2.配置环境变量

打开cmd命令编辑窗口，命令窗口输入perl -v查看环境变量是否配置好了。

!

若配置好了会出现perl的版本等信息，若错误则出现不是内部或外部命令等。

3.修改config.txt配置文件

NOVOPlasty脚本压缩包种提供了默认文件，只需要修改部分参数。

文件内容：

!

自己需要调整的参数：

Project name:自定义项目名称
Type: 如果是组装线粒体需要将这一项改为 mito
Genome Range:鱼类线粒体基因组序列长度通常为16—20kb左右，这一项可以改为16000-20000
K-mer：一般直接用默认的39即可
Max memory：Extended log：无需调整
Save assembled reads：改为yes的会将用于组装的数据以fasta的格式保留下来
Seed Input：种子序列的路径，软件包里提供了一个Seed_RUBP_cp.fasta文件，直接使用就可以
Reference sequence：参考序列的路径，没有参考序列，等号后面的内容需删除
Variance detection：直接空着就可以
Chloroplast sequence：组装线粒体基因组的时候需要把等号后面的内容删掉
Read Length Insert size Platform：这些参数可以在你的测序报告中找到
Single/Paired:选择双向测序数据PE
Combined reads：不用管
Forward reads：第一个fastq文件的路径
Reverse reads：第二个fastq文件的路径

此部分参考：https://www.jianshu.com/p/88e660e46b28

4.运行脚本

当配置文件config参数修改完毕后只需要运行脚本即可。

通过cmd命令编辑窗口，打开脚本即可。

①找到你下载的NOVOPlasty-master文件夹，我这里放到了E盘根目录，这里尽量路径中尽量不要用中文字符。

E：回车——进入E盘

cd NOVOPlasty-master——打开此文件夹

perl NOVOPlasty3.8.3.pl -c config.txt——运行脚本

!

这样就可以等待结果了。（这里只是用原始的config进行展示，也没有数据）
